Defined in: [core/src/types/Document.ts:165](https://github.com/sedrew/vyaz/blob/main/packages/core/src/types/Document.ts#L165)

Numbering format for ordered lists.

- `'decimal'`: 1, 2, 3, ...
- `'upper-roman'`: I, II, III, ...
- `'lower-roman'`: i, ii, iii, ...
- `'upper-alpha'`: A, B, C, ...
- `'lower-alpha'`: a, b, c, ...
