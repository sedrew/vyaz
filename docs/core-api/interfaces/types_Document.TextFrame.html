    Interface TextFrame

    Root text container — a text box on a canvas.

    This replaces the earlier RichTextDocument. It holds geometry, text flow settings, and the paragraph array.

    Wrap & Autofit interaction:

    • When wrap is true, the layout engine breaks lines at width.
    • When autofit.enabled is true, the engine shrinks the font proportionally to fit the text inside width × height.

    Example

    {
      width: 600,
      height: 400,
      wrap: true,
      autofit: { enabled: true, minFontSize: 10, maxFontSize: 24 },
      writingMode: "horizontal-tb",
      verticalAlignment: "top",
      paragraphs: [ /* ... */ ]
    }
