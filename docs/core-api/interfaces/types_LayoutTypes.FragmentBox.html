    Optionaltrailing

    trailing?: boolean

    Признак концевого пробела (trailing whitespace).

    • true: фрагмент находится в конце строки, не участвует в advance строки и не растягивается при justify (нулевая ширина для расчётов).
    • undefined/false: обычный фрагмент.

    См. CSS Text Module Level 3 §4.1.3 (Tracking and Dropping Spaces) и Parley LineItemData::has_trailing_whitespace.
